# Sign Language to Text and Speech Translation in Real Time using CNN

## 📌 Overview
This project is a real-time Sign Language Recognition system that converts hand gestures into text and speech using Computer Vision and Convolutional Neural Network s (CNN). 

## Project Description
This project aims to bridge the communication gap between deaf-mute individuals and non-signers by converting sign language gestures into text and speech in real time using Convolutional Neural Networks (CNN).

## Features
- Real-time hand gesture recognition
- Converts gestures into text
- Converts text into speech
- Webcam-based detection
- User-friendly GUI using Tkinter

## 📸 Working
1. Upload the hand gesture dataset.
2. Train or load the CNN model.
3. Start webcam detection.
4. Show hand gestures in front of the camera.
5. The system predicts the gesture and converts it into text and speech.
